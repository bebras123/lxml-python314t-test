# Just for testing lxml

1. Create sample xml `python sample.py` --> example.xml
2. Clone lxml to this directory: `git clone git@github.com:scoder/lxml.git` 
3. `git checkout ft_python`
4. Run `docker compose build`
5. Run `docker compose run --rm python314-lxml bash`
6. Run `./build_all_lxml.sh`
7. Run `./run_all.sh` for test all builds
or type `py314t` and `python run.py` 

Run logs are stored in `logs` directory
